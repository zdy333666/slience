接口说明



  • 接口名称：G20-应用接口访问统计结果查询

    方法：GET

    URL：/api_count/query

    参数：
    range:起始时间,截止时间

    示例：

    /g20-report/api_count/query?range=2016-08-21 22:00:00,2016-08-22 08:00:00


  • 接口名称：G20-应用接口访问统计结果导出

    方法：GET

    URL：/api_count/export

    参数：
    range:起始时间,截止时间

    示例：

    /g20-report/api_count/export?range=2016-08-21 22:00:00,2016-08-22 08:00:00
